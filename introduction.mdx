Medal Forge is a browser-based workbench that converts uploaded SVG files into 3D medal, badge, and metal plate models. Each SVG shape is parsed into an individual layer that you can extrude, bevel, and texture independently. When you're done, export a GLB file for any 3D pipeline or copy a ready-to-paste React/Three.js component to embed the model directly in your app.

Medal Forge is designed for award design, branded merchandise prototyping, and anyone who needs to visualize flat vector artwork as physical-looking 3D objects without leaving the browser.

## How the workbench works

The workbench opens to a three-panel layout:

- **Layers sidebar (left)** — lists every shape parsed from your SVG. Click a shape to select it; use Cmd/Ctrl-click to add more shapes to the selection; Shift-click to select a range. Click the eye icon to toggle visibility.
- **3D viewport (center)** — renders the live Three.js preview. Orbit, pan, and zoom with your mouse or trackpad. Changes to any setting update the model in real time.
- **Settings panel (right)** — controls for the selected layer (thickness, bevel, material, color, Z-offset) and global canvas settings (background color, grid, shadows, dome effect).

The toolbar at the top of the viewport contains the **Upload** button for importing a new SVG, the **Export GLB** button, and the **Save** control for returning to the home screen.

## Tech stack

Medal Forge is built on:

- **Next.js 16** and **React 19** for the application shell
- **Three.js r184**, **@react-three/fiber**, and **@react-three/drei** for 3D rendering
- **IndexedDB** (via browser-native APIs) for local work persistence

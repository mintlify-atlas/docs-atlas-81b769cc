Medal Forge is a browser-based workbench that transforms uploaded SVG files into fully realized 3D medal and badge models. Upload any SVG, tune each shape's thickness, bevel, material, and color in real time, then export the result as a GLB file ready for use in any 3D pipeline — or embed it directly in a React app with a generated code snippet.

## How it works

<Steps>
  <Step title="Upload your SVG">
    Drag and drop an SVG file onto the workbench or click the upload area. Medal Forge parses every shape layer and renders them as extruded 3D geometry in the live preview.
  </Step>
  <Step title="Tune each layer">
    Select one or more shapes from the layer list. Adjust thickness, bevel, material preset, color, and Z-offset. Changes appear instantly in the 3D viewport.
  </Step>
  <Step title="Apply effects">
    Optionally enable the dome effect to give your medal a spherical curvature. Toggle grid and shadows in the canvas settings.
  </Step>
  <Step title="Export your model">
    Click **Export GLB** to download a binary GLTF file, or copy the React/Three.js code snippet to embed the model directly in your app.
  </Step>
</Steps>

## Embed in your app

Medal Forge can export a ready-to-paste React component that renders your 3D model using `@react-three/fiber`. See the [code snippet reference](/reference/code-snippet) and [export model reference](/reference/export-model) for the full API.
